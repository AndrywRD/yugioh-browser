Adicione aqui os retratos dos NPCs do modo campanha.

Formato recomendado:
- `PNG`
- proporcao aproximada `4:3` (ex.: `512x384`)
- fundo livre (com ou sem transparencia)

Nome do arquivo:
- Use o `id` do NPC retornado pela API de PVE.
- Exemplo: `FM_01_SIMON_MURAN.png`

Fallback automatico:
- Se o retrato nao existir, a UI usa a carta assinatura do NPC como imagem fallback.
