# Yu-Gi-Oh! Súbita - Resumo Completo de Funcionalidades

Este documento resume, de forma simples, tudo o que o projeto já oferece hoje para o jogador.

## 1) Visão geral do jogo

Yu-Gi-Oh! Súbita é um jogo de cartas no navegador com foco em:

- Campanha contra NPCs (PvE)
- Duelo contra outros jogadores (PvP)
- Montagem e evolução de decks
- Descoberta de fusões
- Progressão de conta
- Recursos sociais (amigos, chat e ranking)

## 2) Conta e entrada no jogo

O jogador já consegue:

- Criar conta
- Fazer login
- Manter progresso salvo
- Editar nome de duelista
- Escolher avatar
- Sair da conta

Tudo isso já está integrado ao fluxo do lobby.

## 3) Lobby principal (hub do jogo)

O lobby funciona como central do jogador e já traz:

- Acesso rápido para Campanha, Arena PvP, Deck Builder, Loja e Perfil
- Exibição do deck ativo
- Indicadores de progresso do jogador
- Notificações rápidas no próprio lobby
- Navegação direta para as telas principais

## 4) Campanha PvE

A Campanha já permite:

- Ver NPCs disponíveis e bloqueados
- Iniciar duelo direto com NPC liberado
- Visualizar status de cada NPC (pendente, concluído, bloqueado)
- Ver recompensas de ouro
- Consultar drops e progresso de coleta por NPC
- Receber recompensas após vitória
- Fazer revanche rápida

Também já existe ajuste de dificuldade por nível de oponente, com NPCs mais fortes nos tiers altos.

## 5) Arena PvP (online)

O modo online já possui:

- Lista de jogadores online
- Lista de salas abertas
- Buscar duelo rapidamente
- Criar sala
- Entrar por código
- Entrar em sala disponível

Depois disso, o jogador é levado para a partida.

## 6) Duelo (partida)

No duelo, o jogo já cobre:

- Turnos e fases
- Invocação
- Set de monstro
- Posição de batalha
- Ataque direto e ataque em alvo
- Defesa face-down e revelação
- Uso de magias e armadilhas
- Fluxo de vitória/derrota
- Tela de resultado com recompensas no PvE

Também há efeitos visuais de duelo já integrados no fluxo da partida.

## 7) Deck Builder

O construtor de deck já entrega:

- Visualização da coleção do jogador
- Busca e filtros de cartas
- Adição e remoção rápida no deck
- Controle de quantidade por carta
- Criação, renomeação e exclusão de decks
- Definição de deck ativo
- Validação de deck (ex.: 40 cartas)
- Salvamento persistente

## 8) Fusion Log + Fusion Lab

O sistema de fusão já inclui:

- Histórico das fusões descobertas
- Busca e filtros no log
- Exibição dos materiais usados
- Resultado da fusão
- Laboratório de teste de fusão com 3 slots
- Registro de descobertas novas ao testar

## 9) Loja (Card Trader)

A loja já está funcional com:

- Ofertas rotativas
- Compra de carta unitária
- Atualização de ofertas (reroll)
- Pacotes/boosters
- Preços por valor e progressão do jogador
- Confirmação de compra e feedback visual
- Atualização de gold e coleção após compra

## 10) Perfil, títulos e conquistas

O jogador já pode:

- Ver estatísticas principais (PvE, PvP, cartas, gold, rank)
- Equipar títulos desbloqueados
- Atualizar dados do perfil
- Trocar avatar
- Acompanhar conquistas
- Resgatar progresso de conta de forma clara

## 11) Sistema social (comunidade)

Já existe base social integrada com:

- Lista de amigos
- Solicitações recebidas e enviadas
- Busca de jogadores
- Aceitar/recusar/cancelar solicitações
- Remover amigo
- Conversa privada (chat)
- Contador de mensagens não lidas

## 12) Ranking

O ranking já oferece:

- Ranking global
- Ranking entre amigos
- Atualização de posições
- Destaque do próprio jogador

## 13) Experiência visual e usabilidade

O projeto já conta com:

- Identidade visual temática do jogo
- HUD customizada nas principais telas
- Efeitos visuais de ação no duelo
- Trilha sonora por contexto (lobby, duelo, deck, seleção de NPC)
- Controle de áudio com mute/volume
- Interface adaptada para desktop e evolução para mobile

## 14) Estrutura de progresso do jogador

Hoje o jogo já possui:

- Ouro e recompensas por partida
- Evolução por vitórias
- Coleção persistente
- Decks persistentes
- Conquistas e títulos
- Evolução de conta com continuidade entre sessões

## 15) Estado atual para apresentação

Em resumo, o projeto já está em um estágio jogável e completo para demonstração, com:

- Loop principal funcional (entrar -> montar deck -> jogar PvE/PvP -> evoluir)
- Progressão de conta real
- Economia de loja
- Sistema social
- Ranking
- Duelo com regras e feedback visual

Ou seja: já não é só um protótipo de tela; já é um jogo browser com sistemas centrais conectados.
